Theme Documentation

Complete guide to all available themes in our design system. Each theme features carefully crafted color palettes designed for optimal user experience.

Soft Lavender

UAP Style

Color Palette

Primary Background #faf8ff
Secondary Background #f5f2ff
Primary Color #9b7ed8
Primary Hover #8a6bc7
Primary Text #2d1b4e
Secondary Text #6b5b8a

Description

Soft Lavender is a gentle, soothing theme perfect for creating a calm and peaceful user experience. The light purple tones provide excellent readability while maintaining a modern, elegant aesthetic. This theme is ideal for content-heavy applications and long reading sessions.

Usage

Best suited for:

  • Content platforms and blogs
  • Educational websites
  • Portfolio sites
  • Applications requiring extended reading

UAP Midnight Lavender

UAP Style

Color Palette

Primary Background #1a1526
Secondary Background #241d35
Primary Color #7c5fb8
Primary Hover #8a6bc7
Primary Text #e8dff5
Secondary Text #c4b3d9

Description

UAP Midnight Lavender is a sophisticated dark theme that reduces eye strain in low-light environments. The deep purple backgrounds combined with soft lavender accents create a premium, modern look. Perfect for night-time browsing and professional applications.

Usage

Best suited for:

  • Professional dashboards
  • Developer tools and IDEs
  • Media consumption apps
  • Applications used in low-light environments

Beautiful Cyan
